# PrepMode UI Image References

These images are visual references only.

Claude Code must extract a consistent reusable design system from these images instead of copying inconsistent screenshots blindly.

## Fixed UI Rules

- Use light gray app background.
- Use white raised cards.
- Use indigo/purple as the primary accent.
- Use pale sidebar for learner and admin layouts.
- Use soft lavender active sidebar states.
- Use rounded corners, thin borders, soft shadows.
- Keep sidebar formatting consistent across all screens.
- Do not use dark sidebars.
- Do not include Pro, Upgrade, Premium, Mock Tests, Study Plan, Notes, Community, Leaderboard, AI, Courses, Syllabus, Quant, Reasoning, or unrelated exam modules.
- Replace any inconsistent image copy with the product scope from Summary.md.
