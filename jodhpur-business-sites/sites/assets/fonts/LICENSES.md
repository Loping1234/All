# Instrument Font Assets

The local WOFF2 files in this directory come from:

- `@fontsource/instrument-sans` version 5.2.8
- `@fontsource/instrument-serif` version 5.2.8

Both packages are distributed under the SIL Open Font License 1.1 (`OFL-1.1`).

Sources:

- https://fontsource.org/fonts/instrument-sans
- https://fontsource.org/fonts/instrument-serif

The packages remain listed in `package.json` and `package-lock.json` so the copied assets can be reproduced and audited.
