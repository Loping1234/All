The system is built upon the MERN stack i.e. MongoDB, Express.js, Node.js, and React augmented with Socket.io to facilitate low-latency, bidirectional communication across all platform modules. A central feature of the platform is a real-time Kanban board implemented using the @dnd-kit library, enabling fluid drag-and-drop task management with instantaneous state synchronization across all connected clients, thereby maintaining a consistent single source of truth for distributed teams. The task management subsystem incorporates granular versioning for task metadata, comprehensive audit logging, and threaded comment sections with multi-file attachment support, facilitating context aware collaboration within the task lifecycle.The platform integrates a multi-tiered communication infrastructure encompassing private direct messaging, team-wide broadcast channels, and administrative announcement systems, all equipped with real time typing indicators, read receipts, and message editing capabilities. Presence awareness mechanisms expose live user activity streams and online status indicators, enhancing team transparency and situational awareness. For administrative stakeholders, an analytics suite powered by Recharts delivers interactive, data driven visualizations of task distribution, team performance metrics, and project completion rates. Role based access control (RBAC) governs a hierarchical permission model across Admin, Sub-admin, and Employee roles, enabling fine-grained organizational management.Security is enforced through JSON Web Token (JWT)-based session management, supplemented by optional Two-Factor Authentication (2FA) via email delivered one-time passwords (OTP). Asset management is delegated to Amazon Web Services S3, ensuring scalable and highly available storage for all media and document uploads. The user interface supports configurable dark and light themes and incorporates Three.js-based interactive visual elements. Collectively, TASK-OPS presents a scalable, secure, and feature-rich solution for organizations seeking to manage project lifecycles with precision and collaborative efficiency.	iChapter No.TitlePage No.Abstracti1Introduction11.1Overview of the Internship11.2Objectives of the Internship11.3Scope of the Internship22Company Profile32.1Company Name & Address32.2Nature of Business & Core Services32.3Technologies / Tools Used in the Company43Internship Work Description53.1Department / Team Assigned53.2Projects / Tasks Assigned53.2.1Phase 1 — Backend Fundamentals Training53.2.2Phase 2 — Independent Project: TaskOps53.2.3Phase 3 — Company Product Contribution: TV19 News63.3Learning Outcomes6-73.4Tools, Technologies & Frameworks Used74Gantt Chart (Project Timeline)85Weekly Progress Summary96Future Work & Next Steps106.1Upcoming Project Phases / Tasks106.2Areas for Further Improvement10-117Conclusion127.1Summary of Key Learnings127.2Impact of the Internship on Career Plans12-13Chapter 1: Introduction1.1 Overview of the InternshipThis internship was undertaken as part of an industry exposure program, providing hands-on experience in full-stack web development within a professional software development environment.
